## System Architecture

```
┌────────────────────────────────────────┐
│     User's Web Browser                 │
│  http://localhost:5000                 │
│                                        │
│  [Emergency Alert Button] (RED)        │
└─────────────┬──────────────────────────┘
              │
              ↓
┌─────────────────────────────────────────────┐
│    JavaScript on Frontend                   │
│  triggerEmergency() function                │
│  - Shows confirmation                       │
│  - Sends AJAX POST to /emergency/trigger    │
│  - Shows real-time feedback                 │
└─────────────┬───────────────────────────────┘
              │
              ↓
┌──────────────────────────────────────────────┐
│    Flask Backend (Python)                    │
│  @app.route('/emergency/trigger')            │
│  - Receives POST request                     │
│  - Builds email content                      │
│  - Spawns email thread                       │
│  - Returns JSON response                     │
└─────────────┬────────────────────────────────┘
              │
              ↓
┌──────────────────────────────────────────────┐
│    Email System (Background Thread)          │
│  send_emergency_email()                      │
│  - Connects to SMTP (Gmail)                  │
│  - Creates HTML email                        │
│  - Sends to each recipient                   │
│  - Logs to emergency_alerts.json             │
└─────────────┬────────────────────────────────┘
              │
    ┌─────────┴──────────┬──────────────┐
    ↓                    ↓              ↓
[Students]         [Staff]        [Log File]
get email          get email      saved
